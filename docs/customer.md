# Customer.md

## ICP

- 삼성생명 또는 유사한 보험 영업조직에서 채널별 물량을 관리하는 영업/운영 조직
- 월별 업적 물량, 조직 효율, 신상품, 시책, 시장/경쟁사 이슈를 함께 보고 판단해야 하는 조직
- 임원/유관부서 파트장 보고를 위해 숫자와 해석을 빠르게 정리해야 하는 조직

## 페르소나

### 1. 영업/채널 물량 담당자

- 핵심고통
  - 월별 물량과 채널별 흐름을 여러 파일에서 따로 확인해야 한다.
  - 숫자는 볼 수 있지만 왜 그런 흐름인지 설명하는 데 시간이 많이 든다.
  - 다음달 물량을 예측할 때 사람 경험 의존도가 높다.
- 현재대안
  - 엑셀 수치 직접 조회
  - 회의자료, 시책 문서, 메모를 함께 보며 수기 해석

### 2. 보고자료 작성 실무자

- 핵심고통
  - 숫자만이 아니라 채널별 이슈와 방향성까지 보고 문장으로 정리해야 한다.
  - 같은 데이터를 보고도 설명 문장이 사람마다 달라진다.
  - 보고 시점마다 빠르게 전망과 근거를 다시 만들어야 한다.
- 현재대안
  - 과거 보고서 복사 후 수정
  - 실적표와 이슈 문서를 수작업으로 합쳐 문장 작성

### 3. 유관부서 파트장

- 핵심고통
  - 세부 원자료를 다 읽을 시간이 없다.
  - 총계 숫자, 채널별 방향성, 확인할 리스크만 빠르게 보고 싶다.
  - 전망 숫자가 바뀌었을 때 왜 바뀌었는지 즉시 파악하기 어렵다.
- 현재대안
  - 실무자 브리핑 의존
  - 요약 보고서만 확인
